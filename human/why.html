WHY PZP

The inevitable web.

Machines will consume the web whether we design for them or not.
The question is: badly or well?

The Current State

Today, AI agents scrape human websites. They're remarkably good at it, but it's wasteful and fragile.

What Happens Today What Should Happen
Agent fetches HTML page Agent fetches structured endpoint
Parses navigation, ads, footers Receives only relevant content

Executes JavaScript (maybe) No JavaScript needed
Extracts text with heuristics Consumes native markdown
Wastes 80%+ tokens on cruft 100% tokens are useful

The Numbers

80% Token waste on average

Cloudflare measured 80% token reduction converting their blog from HTML to markdown. Most tokens in HTML are structure, not content.

$0 JavaScript execution in most AI crawlers

ChatGPT, Claude, and Perplexity crawlers do not execute JavaScript. Your SPA is invisible to them.

∞ Sites with different structures

Every website is unique. Every scraping solution is custom. No standard means no ecosystem.

The Inevitability Argument

"Machines will consume the web whether we design for them or not. The question is: Do we want them parsing our human interfaces badly, or consuming our machine interfaces well?"

Three futures exist:

  1. No adaptation. AI agents continue scraping human sites. Wasteful, fragile, expensive. The status quo persists until it doesn't.
  2. Fragmented solutions. Each company builds proprietary machine interfaces. No interoperability. Walled gardens. Racing to the bottom.
  3. Open standard. A shared protocol emerges. Agents can consume any compliant site. Tooling flourishes. Everyone wins.

PZP is option three.

Why Now

Several forces converge:

  • Agent capability. AI agents are now capable enough to consume structured content meaningfully.
  • API costs. Token costs matter. Companies spending millions on AI want efficiency.
  • Existing standards. llms.txt, AGENTS.md, MCP—the ecosystem is ready for synthesis.
  • Critical mass. Enough sites adopting similar patterns to standardize.

The window is open. Standards crystallize now or fragment later.
