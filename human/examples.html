  "name": "Example API Documentation",
  "description": "Complete API reference for Example Platform",
  "type": "documentation",
  "tokenBudgets": {
    "essential": 400,
    "full": 3000
  },
  "capabilities": ["read", "query"]

context/essential.md

# Example API Essential Context

## What This Is

REST API for the Example Platform. JSON responses.
OAuth2 authentication required for all endpoints.

## Key Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /users | List users |
| GET | /users/{id} | Get user |
| POST | /users | Create user |
| DELETE | /users/{id} | Delete user |

## Authentication

Bearer token required. Get token from /oauth/token.

## Rate Limits

- 1000 requests/hour per token
- 429 response when exceeded
